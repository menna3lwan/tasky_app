# Tasky App

Tasky App is a productivity-focused task management mobile application built with **Flutter** and powered by **Firebase**.  
The app helps users organize daily tasks, maintain productivity, and build habits using a dynamic streak tracking feature.

## Features

- User registration & login (Firebase Authentication)  
- Add, update, and delete tasks  
- Task categorization and status tracking  
- Real-time updates with Firebase  
- Streak tracking for habit building  
- Push notifications  
- Intuitive and responsive UI  

---

## Architecture

The application follows **Clean Architecture principles**, ensuring:

- Clear separation of concerns  
- Scalable codebase  
- Easy testability  
- Maintainable layers
